# Olay NameGen

Just a development example.

**Module Handle:** `namegen`

---

## Module Parameters

### updateRate

In which interval the data should be reloaded.

Type: integer (milliseconds)  
Default: `5000`  
Valid: Integers >= 1

### type

Which type of name to generate.

Type: string  
Default: `artist`  
Valid: `artist` | `record`
